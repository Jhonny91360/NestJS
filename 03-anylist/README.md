# Development enviroment

1. Clone project
2. Copy `.env.template` and rename to `.env`
3. Execute
   ```
   pnpm install
   ```
4. Start the image
   ```
   docker-compose up -d
   ```
5. Start backend
   ```
   npm start:dev
   ```
6. Go to
   ```
   localhost:3000/graphql
   ```
